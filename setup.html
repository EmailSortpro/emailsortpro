1 Créer une App Registration Azure

1. Allez sur Azure App Registrations

2. Connectez-vous avec votre compte Microsoft

3. Cliquez sur "New registration"

4. Remplissez le formulaire :

  • Name : EmailSortPro
  • Account types : Personal Microsoft accounts only
  • Redirect URI : Single-page application (SPA)

5. Pour l'URI de redirection, copiez exactement :

2 Configurer les permissions

1. Dans votre app Azure, allez dans "API permissions"

2. Cliquez sur "Add a permission"

3. Choisissez "Microsoft Graph"

4. Sélectionnez "Delegated permissions"

5. Ajoutez ces permissions :

  • ✓ User.Read
  • ✓ Mail.Read
  • ✓ Mail.ReadWrite
