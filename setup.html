1 Mettre à jour votre App Registration Azure

1. Allez sur Azure App Registrations

2. Trouvez votre application EmailSortPro existante OU créez-en une nouvelle

3. Allez dans "Authentication"

4. IMPORTANT: Mettez à jour les URI de redirection avec les nouvelles valeurs :

URI de redirection (Single-page application):
https://emailsortpro.netlify.app/auth-callback.html
URI de déconnexion:
https://emailsortpro.netlify.app/

Note: Supprimez les anciennes URI si elles pointent vers un autre domaine.

2 Vérifier les permissions API

1. Dans votre app Azure, allez dans "API permissions"

2. Vérifiez que ces permissions sont accordées :

  • ✓ User.Read (Microsoft Graph)
  • ✓ Mail.Read (Microsoft Graph)
  • ✓ Mail.ReadWrite (Microsoft Graph)

3. Si manquantes, cliquez sur "Add a permission" → "Microsoft Graph" → "Delegated permissions"
